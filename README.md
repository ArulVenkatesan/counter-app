🧮 Counter App
📌 Overview
A simple React application that demonstrates the use of state management and conditional rendering.
The app allows users to increment, decrement, and reset a counter value, with logic to prevent the counter from going below zero.

🚀 Features
- Increment button ➝ increases the counter value by 1
- Decrement button ➝ decreases the counter value by 1 (but never below 0)
- Reset button ➝ resets the counter back to 0
- Responsive card layout styled with TailwindCSS
🛠️ Tech Stack
- React – for building the UI and managing state with useState
- TailwindCSS – for styling and responsive design

📖 What I Learned
- How to use the useState hook to manage component state
- Implementing conditional rendering to prevent negative values
- Structuring a simple React component with reusable UI elements
- Applying TailwindCSS utility classes for quick styling
